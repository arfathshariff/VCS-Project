  # Code of Conduct

## 1. Be Respectful  
Treat everyone with respect. No harassment, derogatory comments, or personal attacks.  

## 2. Be Inclusive  
We welcome people from all backgrounds, identities, and levels of experience. Make sure your language is friendly and constructive.

## 3. Use Clear Communication  
Use understandable language. Avoid jargon if possible. Be open to feedback and willing to explain your reasoning.

## 4. Collaborate Honestly  
When contributing, be honest about your work. If you borrowed code or ideas, give credit.  

## 5. Resolve Conflicts Constructively  
If disagreements arise, discuss them calmly. Focus on finding a solution rather than blaming.

## 7. Report Issues  
If someone violates this code of conduct, report it to the project maintainers. We will handle it fairly and confidentially.
